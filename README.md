#  Data Analytics Visualization Project

This project showcases my data analytics and data visualization skills using Python (pandas and matplotlib) in Google Colab.  
The analysis is based on a customer support dataset to uncover trends, performance insights, and workflow patterns.

---

##  Project Files

| File | Description |
|------|-------------|
| **customer_support_dataset.csv** | Dataset used for the analysis |
| **analysis.ipynb** | Jupyter Notebook with Python code and visualizations |
| **tasks_by_type.png** | Bar chart showing number of tasks by category |
| **tasks_per_day.png** | Line chart showing tasks trend over time |
| **task_status.png** | Pie chart showing task completion status |
| **tasks_by_hour.png** | Bar chart showing busiest hours of the day |
| **response_vs_rating.png** | Scatter plot showing relationship between response time and customer rating |

---

## Visualizations Included

- **Tasks by Type** — Which types of requests happen the most  
- **Tasks per Day** — Daily trends and workload patterns  
- **Task Status Distribution** — Completion performance  
- **Busy Hours** — Time of day with most activity  
- **Response Time vs Customer Rating** — Performance impact visualization
